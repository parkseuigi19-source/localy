## 🌟 방법 1: MoviePy + AI 텍스트 (추천!)

### 특징
- ✅ **AI가 사진 분석**: 각 사진의 내용을 GPT-4 Vision이 자동 분석
- ✅ **감성적인 텍스트**: 사진마다 어울리는 텍스트 자동 생성
- ✅ **고화질**: CRF 18, 8000k bitrate로 거의 무손실 수준
- ✅ **빠른 생성**: 사진 10장 기준 1-2분 내 완료
- ✅ **저렴한 비용**: 사진 10장에 약 11원

### 필요한 API
- **OpenAI API**: https://platform.openai.com/api-keys
- **모델**: GPT-4o (Vision)

### 설정 방법
```bash
# 1. .env 파일 생성
OPENAI_API_KEY=sk-proj-xxxxxxxxxxxxx

# 2. 사진 준비
# photos 폴더에 사진 넣기

# 3. 실행
python create_reels_simple.py
```

### AI 텍스트 스타일

#### 1. Poetic (감성형) ⭐ 기본값
사진의 분위기를 감성적으로 표현
```python
ai_text_style="poetic"
```
**예시:**
- "황금빛 추억"
- "별이 빛나는 밤"
- "바다와 함께하는 여름날"
- "행복한 웃음소리"

#### 2. Descriptive (설명형)
사진의 주요 내용을 간결하게 설명
```python
ai_text_style="descriptive"
```
**예시:**
- "해변의 석양"
- "도심 야경"
- "맛있는 음식"

#### 3. Simple (간결형)
짧고 간단한 단어로 표현
```python
ai_text_style="simple"
```
**예시:**
- "여유로운 오후"
- "특별한 하루"
- "평화로운 시간"

## ⚡ 방법 2: MoviePy 기본 (완전 무료)

### 설정 방법
```python
# create_reels_simple.py에서 AI 비활성화
config = ReelsConfig(
    enable_text_overlay=True,      # 텍스트 오버레이 활성화
    enable_ai_text_overlay=False,  # AI 비활성화 → 날짜 표시
    # ... 기타 설정
)
```

### 실행 방법
```bash
python create_reels_simple.py
```

### 사용 가능한 효과

#### 1. Ken Burns 효과
```python
config = ReelsConfig(
    enable_ken_burns=True,
    ken_burns_style="random",  # zoom_in, zoom_out, pan_left, pan_right, diagonal
    effect_intensity="medium"  # low, medium, high
)
```

#### 2. 전환 효과
```python
config = ReelsConfig(
    enable_transitions=True,
    transition_style="random"  # fade, slide, zoom, random
)
```

#### 3. 회전 효과
```python
config = ReelsConfig(
    enable_rotation=True  # 미묘한 회전 (-3도 ~ +3도)
)
```

#### 4. 텍스트 오버레이
```python
config = ReelsConfig(
    enable_text_overlay=True  # 촬영 날짜 표시
)
```

## 🎨 화질 설정

### 현재 적용된 고화질 설정
```python
# reels_engine.py에 자동 적용됨
bitrate='8000k',           # 높은 비트레이트
preset='slow',             # 더 나은 압축 품질
crf='18',                  # 거의 무손실 수준 (0-51, 낮을수록 고화질)
profile='high',            # H.264 High Profile
pix_fmt='yuv420p'          # 호환성
```

### 화질 vs 속도 조절
인코딩 속도를 높이려면 `preset` 변경:
- `ultrafast`: 가장 빠름, 낮은 품질
- `fast`: 빠름, 중간 품질
- `medium`: 보통 속도, 좋은 품질
- `slow`: 느림, 매우 좋은 품질 ⭐ 현재 설정
- `veryslow`: 매우 느림, 최고 품질

```bash
# 1. API 키 확인
https://platform.openai.com/api-keys

# 2. 크레딧 확인
https://platform.openai.com/usage

# 3. .env 파일 확인
cat .env
```

### Q: AI 텍스트가 생성되지 않아요
```bash
# 콘솔 출력 확인
[AI] 이미지 분석 중: photo1.jpg
[AI] 생성된 텍스트: 황금빛 추억

# 위와 같은 메시지가 보이지 않으면:
# 1. enable_ai_text_overlay=True 확인
# 2. OPENAI_API_KEY 확인
# 3. 인터넷 연결 확인
```

### Q: 텍스트 스타일을 변경하고 싶어요
```python
# create_reels_simple.py 수정
config = ReelsConfig(
    enable_ai_text_overlay=True,
    ai_text_style="descriptive",  # poetic → descriptive로 변경
    # ...
)
```
